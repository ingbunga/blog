어떻게 $Unif(0, 1)$ 값을 생성하는 함수를 가지고 있을때, 순열을 어떻게 랜덤하게 셔플할 수 있을까?

# 알고리즘
1. 어떤 순열 $P_1, P_2, \dots P_n$ 이 있다고 하자.
2. k = n 으로 두고 시작한다.
3. $U \sim Unif(0, 1)$ 일때 $I=\lceil kU \rceil$ $U$를 생성해서 $I$를 구한다.
4. $P_i$와 $P_k$를 바꾼다.
5. $k := k - 1$ 을 하고, 만일 $k > 1$라면,  3으로 돌아간다. 아니라면 멈춘다.
# 코드
```python
import random

r = random.random

def random_shuffle(lst):
    n = len(lst)
    tmp = lst[:]
    for i in range(n, 1, -1):
        j = int(r() * i)
        tmp[i-1], tmp[j-1] = tmp[j-1], tmp[i-1]

    return tmp

l = list(range(10))
print(random_shuffle(l))
```
